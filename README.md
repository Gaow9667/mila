# 智教·极速教学反思生成器 (MVP)

这是为你量身定制的“教学反思生成器”最小可行性产品 (MVP)。

## 核心功能
*   **极简输入**：通过勾选“亮点”和“不足”，代替繁琐的打字。
*   **专业话术**：内置资深教师 Prompt，生成符合公立学校检查标准的公文风反思。
*   **模型兼容**：默认适配 DeepSeek，也支持任何兼容 OpenAI 接口的模型。

## 如何运行

1.  **安装环境**
    确保你已经安装了 Python。然后打开终端，运行以下命令安装所需库：
    ```bash
    pip install -r requirements.txt
    ```

2.  **启动应用**
    在终端中运行：
    ```bash
    streamlit run app.py
    ```

3.  **使用**
    *   浏览器会自动打开 http://localhost:8501
    *   在左侧边栏输入你的 API Key (推荐 DeepSeek)。
    *   选择课程信息，勾选标签，点击“生成”。
